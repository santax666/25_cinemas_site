Для удобства использования на сайте применяется публичное HTTP API, предоставляющее информацию о фильмах в формате JSON.

На данный момент реализовано 3 варианта GET-запросов к API:

schedule

В результате запроса предоставляется краткая информация о фильмах, которые показывают сегодня в кинотеатрах.

У запроса есть необязательный параметр - date, со следующими значениями:

  • 1: Сегодня
  • 2: Завтра
  • 3: Выходные
  • 4: Следующая неделя
  • 5: Следующие выходные
  • 6: Ближайший месяц

Примеры запросов:

  • /api/schedule - Фильмы сегодня
  • /api/schedule?date=3 - Фильмы в выходные

film

В результате запроса предоставляется полная информация о фильме, включая ссылки на фото и трейлеры.

У запроса есть обязательный параметр - id фильма, значение которого можно узнать либо на сайте Афиши или данном сайте, либо в ключе 'ObjectID' через запрос schedule или search

Примеры запросов:

  • /api/film?id=167945 - Фильм "Такси"
  • /api/film?id=169399 - Фильм "Вий"

search

В результате запроса выполняется поиск и выводится результат о фильмах, которые соответствуют названию, введенному вами.

У запроса есть обязательный параметр - name, являющийся фразой для поиска

Примеры запросов:

  • /api/search?name=matrix - поиск по фразе "matrix"
  • /api/search?name=гараж - поиск по фразе "гараж"
